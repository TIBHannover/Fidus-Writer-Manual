Adding users

Before starting it is important to note that the site owners can setup user account additions to work in two ways:

  1. Where users can register on the site,

  2. Alternatively, where uses cannot register and instead must be invited to the site.

There are different ways to add users to Fidus Writer:

  1. Bulk addition as a Super user from /admin/auth/user/add/. Here a user can be added with: username, password, email, name, given a user role, etc.
